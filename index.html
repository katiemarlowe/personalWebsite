Projects



GiverGram

Making gift giving easier for grandmothers

GiverGram

GiverGram was a group project I made for my User Interface Design class (6.813). Check it out here.


My group and I discovered that while grandmothers greatly enjoy giving their family members gifts, it can often be a difficult task for them. Most do not take advantage of technology to help them keep track of wishlists or events and birthdays. GiverGram attempts to solve this problem by providing an interface that is easy to use and not intimidating.

UglyFruit

Reducing food waste to help end hunger.

UglyFruit

UglyFruit was a group project I made for my Data Storytelling class (CMS.631). Check it out here.


The goal of this project was to convey how sizable of a problem hunger in this country is, and how food waste, which is also a huge problem that we all contribute to, is a possible solution to the problem. We want viewers to understand that while we, as individuals, waste a lot of food, grocery stores are also a big contributor to this problem. Lastly, we want viewers to understand that there are things that can be done about this, which are listed in our call to action section.
